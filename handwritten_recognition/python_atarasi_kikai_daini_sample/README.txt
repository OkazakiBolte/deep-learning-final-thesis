●付属データについて
付属データには『Pyhonで動かして学ぶ！ あたらしい機械学習の教科書 第2版』（ISBN978-4-7981-5991-1）の本文で解説したコードと実行結果のサンプルファイル（第1章から第9章）を用意しています。
サンプルファイルはJupyter Notebook形式です。
Jupyter Notebookのインストール方法は本文の第1章を参照してください。

・Jupyter Notebookへのサンプルファイルの追加方法
Jupyter Notebookを起動すると、［Files］タブの画面（Windows のエクスプローラーが起動）が開きます。
［Files］タブの画面にサンプルファイルをドラッグ＆ドロップして追加できます。
追加したら［Upload］ボタンをクリックします。
アップロードしたファイルをクリックすると編集画面が開きます。
［In］のボックスからコードをコピーできます。またファイル上で、実行結果も確認できます。

●本書のサンプルの動作環境
　本書のサンプルは以下の環境で、問題なく動作することを確認しています。

OS：Windows 10

Anaconda：4.6.11
※本書で利用した「Anaconda3-2019.03-Windows-x86_64.exe」は https://repo.anaconda.com/archive/よりダウンロードできます。
Python：3.7.3
Jupyter Notebook：5.7.8
TensorFlow：1.13.1
Keras：2.2.4

●バージョンを指定したインストール方法
上記のTensorFlowおよびKerasのバージョンを指定してインストールする際は、本書のP.19、P.20でも解説しているようにバージョン指定のコマンドを付けてインストールしてください。

pip install tensorflow==1.13.1

pip install keras==2.2.4
